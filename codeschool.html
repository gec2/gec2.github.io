CURRICULUM HIGHLIGHTS

BECOME A PROFESSIONAL WEB DEVELOPER


HTML & CSS

HTML is the structure that supports the web. You'll use the popular SASS and LESS extensions to CSS to create efficient and organized front-ends.

SysOps

Every developer should be a basic system operator. So you'll know how to setup a cloud server on AWS, deploy to Heroku, and SSH your way around any shell.

JavaScript

You'll bring the web to life with JavaScript and learn new JS libraries like Angular.js, Ember.js, or JQuery.

Ruby on Rails

The number one web framework in use by today's fastest growing tech companies. Web development is now a productive pleasure. You will love Rails and know who DHH is.

Bash / Terminal

Every programmer feels comfortable working on the command line. You'll become just as comfortable navigating files, databases, and the rails console from the terminal.

HTTP Protocol

HTTP is the foundation of data communication for the World Wide Web. Yes, you'll learn to code but you will understand how the Internet is a tool of creation versus a medium for consumption.

Database

Relational or Document Store? MySQL or Postgres? Mongo or Redis? Cloud or dev/null? Learning the fundamentals to data storage is a must, and yes, you will know SQL.

Git Version Control

Your code will be safely tucked away into Git and you'll collaborate with the world through Github. Tame the rainbows and work fluidly with teams, both remote and onsite..
